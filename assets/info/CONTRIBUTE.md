# Contributing.
When contributing at JetTypes, we expect you to follow some rules such as the **CONTRIBUTING.md** file and the **CODE_OF_CONDUCT.md** file.

# Doing A Commit

Before making a Pull request, make sure to:
 - Check for AI Code
 - Check for Vulnerabilities.
 - Follow the rules
 - Test.
 - 
Your request will be checked later, please have paciente if you ever do a pull-request. It may take up to 72 hours for your Commit to be checked.
