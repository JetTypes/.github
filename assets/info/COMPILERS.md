## Our Compilers
We use Maven & Gradle for most projects, although there may be exceptions within the repositories. Our standard packaging format is .jar.

When using our repositories, take in mind most of them are configured to use Java 17 and above. For older versions of Java please change the settings on the respective files.

## Compiling
For compiling, use the respective commands for each Software.

> **Maven**
>  ```
>  mvn clean package
>  ```
> **Gradle**
>  ```
>   gradle build
>  ```
